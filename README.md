# Math Foundations

A structured reference for undergraduate mathematics — from Class 11 through B.Sc. Semester 8.

Built for myself. Useful for anyone studying pure or applied mathematics at the undergraduate level, especially under Indian university syllabuses (SKMU, other state universities).

---

## What this is

A personal knowledge repository containing:
- Core formulas with one-line intuitions
- Key definitions in plain language
- Theorems with brief proof sketches where useful
- Connections between topics across semesters

This is not a textbook replacement. It is a second brain — built to make recall fast and understanding deep.

---

## Who this is for

- B.Sc. Mathematics students (especially Sem 1–8 under NEP 2020)
- Anyone who learns by understanding structure, not memorizing isolated facts
- People preparing for competitive exams, vivas, or research

## Philosophy

> Intuition first. Formalism second. Connection always.

Every formula here comes with the idea behind it — not just the symbol. The goal is to understand why something is true, not just that it is.

This approach is inspired by how working mathematicians actually think — as described by professors at institutions like ISI — where building intuition matters more than brute-force memorization.
