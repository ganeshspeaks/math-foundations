# 1b. Abstract Algebra

## Group Axioms
A set G with operation · is a group if:
1. **Closure**: a·b ∈ G
2. **Associativity**: (a·b)·c = a·(b·c)
3. **Identity**: ∃e such that a·e = e·a = a
4. **Inverse**: ∃a⁻¹ such that a·a⁻¹ = a⁻¹·a = e

## Key Group Formulas
- Order of cyclic group generated by a: |⟨a⟩| = order of a
- Lagrange's Theorem: |H| divides |G| for subgroup H ≤ G
- Number of cosets of H in G: |G|/|H| (index of H)
- For cyclic group of order n: every divisor d of n gives exactly one subgroup of order d
- a^|G| = e for all a ∈ G (consequence of Lagrange)

## Ring Axioms
A set R with + and · is a ring if:
1. (R, +) is an abelian group
2. · is associative
3. Distributive: a·(b+c) = a·b + a·c

## Field
A ring where every nonzero element has a multiplicative inverse.

## Integral Domain
A commutative ring with no zero divisors:
- a·b = 0 ⟹ a = 0 or b = 0

## Homomorphism
- φ: G → H is homomorphism if φ(a·b) = φ(a)·φ(b)
- Kernel: ker(φ) = {a ∈ G : φ(a) = e}
- Image: im(φ) = {φ(a) : a ∈ G}
- First Isomorphism Theorem: G/ker(φ) ≅ im(φ)
